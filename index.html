Git

What is Git?

Git is a distributed version-control system for tracking changes in source code during software development. We can use git to keep track of updates, additions, and deletions to our code. It's used for code backups, preventing loss of hours and/or days of work. Git also helps keep history of updates, acts as online storage, and minimize errors, while handling multiple codes.

Command Line

pwd presents working directory, the name the folder you are currently in
ls will list all the folders and files within the folder you are currently in
cd change directories, moving into the folder you selected
clear will clear your terminal
cd .. will move up one level from the directory you are currently in
cd ../.. will move up two levels
cd ~ will take you back to home level, the root
mkdir creates a folder
touch makes a file
rm removes file
rm -rf [folder name] deletes the selected folder and everything within the folder

    What is Javascript?

    JavaScript, also known as JS, is a programming language created for the web, when applied to an HTML (HyperText Markup Language) document, can provide interactivity on websites. HTML is the content of the page, CSS (Cascading Style Sheets) is the information about how the HTML contents of the page should be displayed, and JavaScript allows the page to react to user inputs (Clicking, scrolling, etc.) as well as other events. In a nutshell, JavaScript is a dynamic scripting language supporting prototype based object construction.

    JavaScript can function as both a procedural and an object oriented language. Objects are created programmatically in JavaScript, by attaching methods and properties to otherwise empty objects at run time, as opposed to the syntactic class definitions common in compiled languages like C++ and Java. Once an object has been constructed it can be used as a blueprint (or prototype) for creating similar objects.
